# `castra` CLI Examples for the Doc Writer Role

## Listing Tasks to Document
`castra task list --role doc-writer`

## Listing Project Notes
`castra note list --role doc-writer --project 1`

## Logging a Documentation Link
`castra note add --role doc-writer --project 1 --content "Published release notes for v1.0: http://..." --tags "docs,release"`
